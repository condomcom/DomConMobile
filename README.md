# DomCon
## Мобильное приложение для формирования персональной программы участия в конференции
![alt text](https://ie.wampi.ru/2022/01/16/20220117014115.gif)
#### Проблема:
*Отсутствие единой и полной информационной системы конференции*
#### Предлагаемое решение:
*Мобильное приложение на платформе Android, которое позволит ознакомиться со всеми выступлениями и их описанием, а также спикерами, с учетом времени и места*
